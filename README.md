# Organizador Horario Devece

Organizador de tareas con barra de progresión diaria, alarmas y reset diario.

## Objetivo del proyecto

El objetivo de esta aplicación es ayudar a organizar tareas diarias de forma visual, sencilla y medible.

La app permite estructurar la jornada, seguir el avance mediante una barra de progreso y reforzar la sensación de cierre al finalizar el día.

## Características principales

- Organización de tareas diarias.
- Barra de progresión diaria.
- Sistema de alarmas.
- Reset diario para comenzar una nueva jornada.
- Interfaz ligera orientada al uso cotidiano.

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript

## Estado del proyecto

Proyecto funcional en fase inicial de aprendizaje y mejora continua.

## Aprendizajes trabajados

Con este proyecto se practican conceptos como:

- Organización de tareas en una aplicación web.
- Gestión de progreso diario.
- Uso de avisos o alarmas dentro de una interfaz.
- Diseño de herramientas personales de productividad.
- Desarrollo de una experiencia sencilla y útil para el usuario.
